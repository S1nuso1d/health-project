# Анализ факторов, влияющих на здоровье

Проект для курсовой работы по теме "Разработка мобильного приложения для анализа здоровья на основе методов машинного обучения".

## О проекте

Проект анализирует данные о здоровье сна и образе жизни для выявления ключевых факторов, влияющих на расстройства сна. Используются методы машинного обучения для прогнозирования типа расстройства сна на основе анкетных и физиологических данных.

## Цели проекта

1. Провести комплексный анализ данных о здоровье сна
2. Сравнить эффективность различных алгоритмов машинного обучения
3. Выявить ключевые факторы, влияющие на качество сна
4. Подготовить основу для разработки мобильного приложения
